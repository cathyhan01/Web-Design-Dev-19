Background info

For this assignment you will need to refer to both the HTML and the JavaScript files, but you will only need to alter the JavaScript.

Let's take a look closely at what's inside our HTML file. Take inside the class .grid. These are tiles with class grid-cell that represent the sounds that we play. Note the id's like brass-soundtile --> they're connected with what sound we want to play.

Now scroll to the bottom of the HTML file. We have a bunch of audio tags with sound files we want to play. Also note the ids. Do they seem familiar?

Let's get familiar with a few new things in our JavaScript file. Take a look at our arrays: keys, sounds, and colors. Note how they all have 9 items in them, one for each tile we see on the screen. The idea is that, when we press a key, we want to play a sound, and flash a color.

To achieve this effect, we have two event handlers: onkeydown and onkeyup. This handles what happens when we press down on a key, and lift up from a key, respectively.

Instructions

Let's create our DJ app by playing a sound inside our onkeydown function! Note that onkeydown takes a parameter called event. We console logged it for you, so check your inspector in your browser and see what it prints!

From there let's think about what we should do for these arrays. Look again above to see what we want to do when we press a key. If we look at our grid and match the arrays keys, sounds, and colors, we see that they are already in order for us! That's great!

We want to do the following

  1. Get and store the index of the key that we pressed (what's event.key?) using .indexOf(). This is a new function that finds the index of the element that we pass in.
  2. Use this index to get the corresponding sound in sounds and color in colors. Call these values soundId and colorId
  3. Inside our conditional statement, we haven't assigned values to sound or tile. Let's use document.getElementByIdto find the right id for our sound tag and .grid-cell element. Do you remember how they're related?
  4. If you got that working, everytime you press a key, it plays!
  5. Change the background-color of tile to colorId.

Mission accomplished! But you might have noticed that once we press a key, the color doesn't change back. Let's change that.

What we want to do is that when we let go of the key, all the tiles return to the color #333333.

  1. In our onkeyup function, let's use document.getElementByClassNames to select all elements with class .grid-cell
  2. Use a for loop to go through each element (how many are there?) using .length. Inside our for loop, set each element's background color back to black.
